QRCode.js, your go to qrcode library

QRCode.js is a JavaScript library designed for generating QR codes efficiently. It also has wide cross-broswer compatibility. Notably, QRCode.js operates independently, devoid of any external dependencies.
So this brief tutorial will guide you on how to use the library for the possible use cases. The library is open source and is hosted on github
